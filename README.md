# Tableau Analytics Showcase

A comprehensive Tableau project demonstrating data visualization and analytics capabilities across Airbnb and Supply Chain datasets.

## Project Overview

This project consists of two main analytical components:

### Part 1: Airbnb NYC Analysis
- Average listing prices by neighborhood
- Occupancy rate analysis by room type
- Geographic and temporal patterns

### Part 2: Supply Chain Analytics
- Logistics performance analysis
- Sales and revenue insights
- Customer geographic distribution
- Predictive analytics and forecasting

## Technologies Used
- Tableau Public
- Data Visualization
- Statistical Analysis
- Predictive Analytics
